# 如何获取阿里云通义千问 (DashScope) API Key

阿里云通义千问（Qwen）提供**免费试用额度**，且兼容 OpenAI 格式，非常适合学生项目。

## 步骤 1：注册/登录阿里云
1.  访问 [阿里云百炼平台 (DashScope)](https://bailian.console.aliyun.com/)。
2.  使用您的阿里云账号登录（如果没有，可以用支付宝/淘宝账号直接登录）。

## 步骤 2：开通 DashScope 服务
1.  登录后，系统可能会提示您**开通 Model Studio (百炼)** 服务，点击确认开通。
2.  **注意**：新用户开通后通常会获得**免费的 Token 额度**（有效期内免费，或有一定数量限制）。

## 步骤 3：创建 API Key
1.  在左侧菜单栏找到 **“API-KEY 管理”** (API Key Management)。
2.  点击右上角的 **“创建新的 API-KEY”**。
3.  系统会生成一个以 `sk-` 开头的密钥。
4.  **复制并保存**这个 Key，它只会显示一次！

## 步骤 4：在项目中使用
1.  打开本项目的网页端。
2.  在侧边栏设置中：
    *   **选择模型厂商**: `Aliyun (Qwen)`
    *   **API Base URL**: 保持默认 (`https://dashscope.aliyuncs.com/compatible-mode/v1`)
    *   **API Key**: 粘贴刚才复制的 `sk-xxxx`
3.  开始对话。

## 常见问题
*   **Q: 需要实名认证吗？**
    *   A: 是的，使用阿里云服务通常需要完成个人实名认证。
*   **Q: 免费额度用完了怎么办？**
    *   A: 通义千问的价格非常便宜（相比 GPT-4 便宜 90% 以上），如果免费额度用完，可以充值几块钱，足够用很久。
