## 任务说明
1.  **项目目标**：在 **2周** 内快速构建一个可用的智能研报助手原型，实现“对话-搜索-导出”核心闭环。
2.  **技术要求**：
    *   **后端**：Python, LangGraph, FastAPI (轻量级API)
    *   **前端**：Streamlit (低代码快速开发)
    *   **搜索**：Tavily API
    *   **文档**：ReportLab
3.  **考核要求**：
    *   重点考察核心功能的实现与运行，界面美观度不作过高要求。
    *   文档需反映敏捷开发的特点，强调核心逻辑与测试结果。
    *   系统需在本地环境稳定运行。
