# 用户手册 (User Manual)

欢迎使用 **智搜研报助手 (Lite 版)**！
本工具旨在帮助您快速进行信息检索、整合，并生成专业的研报 PDF。

## 1. 快速开始

### 第一步：启动服务
如果您是开发者或在本地运行，请确保已按照 `README.md` 启动了后端和前端服务。
*   打开浏览器访问：`http://localhost:8501`

### 第二步：配置 API Key
在左侧侧边栏 (Sidebar) 进行必要的配置：
1.  **选择模型厂商**：支持 OpenAI、DeepSeek、Moonshot (Kimi)、Aliyun (Qwen)。
2.  **填写 API Key**：
    *   **Model API Key**：对应厂商的 Key（如 `sk-...`）。
    *   **Tavily API Key**：用于联网搜索的 Key（`tvly-...`）。
    *   *提示：如果没有 Key，请参考项目文档中的《获取阿里云API_Key教程.md》或《OpenAI额度与替代方案.md》。*

---

## 2. 核心功能使用指南

### 2.1 智能对话与搜索
在页面底部的输入框中输入您的问题。
*   **普通闲聊**：如“你好”、“帮我写一段Python代码”。系统会直接快速回复。
*   **联网搜索**：如“**今天北京天气怎么样？**”、“**2025年AI Agent的发展趋势**”。
    *   系统会自动识别需要搜索，并显示 `🌐 正在联网搜索相关信息...`。
    *   搜索完成后，系统会综合多条结果，为您生成一篇带有引用信息的回答。

### 2.2 导出 PDF 研报
当您完成了一组对话（例如调研了某个行业），可以将其导出为 PDF。
1.  在左侧侧边栏找到 **“📥 导出报告”** 区域。
2.  输入 **报告标题**（默认为“我的研报”）。
3.  点击 **“生成 PDF”** 按钮。
4.  稍等片刻，按钮下方会出现 **“点击下载 PDF”**，点击即可保存文件。
    *   *注：PDF 支持中文显示，包含完整的对话问答记录。*

### 2.3 清空对话
如果您想开启一个新的话题（例如从“天气查询”切换到“论文写作”）：
*   点击左侧侧边栏的 **“🧹 清空对话”** 按钮。
*   这会重置当前的会话历史，避免旧的上下文干扰新话题。

## 3. 常见问题 (FAQ)

**Q: 为什么提示 "API Key 无效"？**
A: 请检查 Key 是否复制完整（不要有多余空格），并确认您选择的“模型厂商”与填写的 Key 是匹配的。

**Q: PDF 下载后打开是空白或者乱码？**
A: 本版本已修复中文乱码问题。请确保后端服务（Backend）正常运行且 `assets` 目录下存在字体文件。如果遇到问题，请尝试刷新页面重试。

**Q: 搜索速度很慢？**
A: 联网搜索取决于 Tavily 接口响应速度和您的网络环境，通常需要 5-15 秒，请耐心等待。

**Q: 刷新页面后历史记录还在吗？**
A: 不在。为了保护隐私并简化设计，本 Lite 版采用内存存储，刷新页面会重置所有会话。请在刷新前及时导出 PDF。
